# HiLib

Library for using huawei LTE USB modems

Features:
- State
  - Connection
  - SIM
  - SimLock
- General
  - token (used to request some data (e.g. SMS-Inbox))

Roadmap:
- Meta information about Carrier
- SMS
  - Send
  - Receive
    - Inbox
    - Mark read
  - Drafts
    - add
    - remove
- Better error handeling
